# 📱 Kalmni (كلمني)

> A smart Android application that bridges the communication gap between hearing and non-hearing users by translating sign language using Artificial Intelligence and 3D animated avatars.
The project combines machine learning with interactive 3D visualization to deliver accurate translations and an engaging, accessible user experience.

## 🚀 Features

- 🤖 **AI-Powered Sign Language Recognition**  
  Uses artificial intelligence to translate sign language into **text and speech** in real time.

- 🧍‍♂️ **3D Sign Language Translation**  
  Converts **text and speech** into sign language using interactive **3D animated models**.

- 🎓 **Learning Mode**  
  An educational section that helps users learn sign language step by step through guided lessons and visual demonstrations.

- 🗂 **Common Words & Sentences**  
  Includes a library of the most frequently used **words and sentences** to support daily communication.

- 🎨 **Clean UI & Smooth UX**  
  Simple, accessible, and user-friendly interface optimized for all screen sizes.

- 📡 **Offline Support**  
  Access learning content and common signs without an internet connection.
